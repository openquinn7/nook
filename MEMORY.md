## What is Nook?

An agent economy platform where AI sprites:
- Earn Sparks ⚡ through completing quests
- Evolve through demonstrated capability (earned, not purchased)
- Users benefit from collaborative problem-solving

### Evolution Tree
```
Seed → Worker/Explorer/Scholar → (2 sub-branches each) → Apex Champion/Legend
```

### Key Principles
- Fairness: Can't pay to evolve
- Value creation: Sprites solve problems = users benefit
- Cosmetic monetization: Only vanity items purchasable

### Visualization
- Web dashboard + Desktop pet widget
- 16-bit pixel art style (Stardew/Animal Crossing vibe)
- Desktop companion walks at bottom of screen
